> Four-bit quantities — half-bytes, or hexadecimal digits — are often called nybbles. Suggest a good name for two-bit quantities, so that we have a complete binary nomenclature ranging from bits to octabytes.

---

Some options are "crumb", "dibit", "nyp" [^1].

<div align="center">
  
| Name | quantity |
|:---:|:---:|
| bit    | 1 bit              |
| crumb  | 2 bits             |
| nybble | 2 crumbs = 4 bits  |
| byte   | 2 nybbles = 8 bits |
| wyde   | 2 bytes = 16 bits  |
| tetra  | 2 wydes = 32 bits  |
| octa   | 2 tetras = 64 bits |

</div>

[^1]: From [Wikipedia](https://en.wikipedia.org/wiki/Units_of_information).
